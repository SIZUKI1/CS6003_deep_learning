# CS6003 Deep Learning & Spatial Intelligence

This repository contains my coursework and assignments for the CS6003 Deep Learning course.

## Directory Structure

* **`HW1/`**: Implementing a 3-layer MLP from scratch (NumPy only) for EuroSAT land cover classification.
* **`HW2/`**: Midterm Assignment focusing on core Computer Vision tasks using PyTorch.
  * **`task1/`**: Fine-tuning CNNs (ResNet-18) on the Oxford 102 Flower Dataset. Includes ablation studies on learning rates, pre-training, and attention mechanisms (SE-block, CBAM).
  * **`task2/`**: Traffic Scene Object Detection & Video Multi-Object Tracking. Fine-tuned YOLOv8s on the Road Vehicle Images Dataset and implemented an inference pipeline with ByteTrack for trajectory tracking, occlusion ID jump analysis, and virtual line-crossing vehicle counting.
  * **`task3/`**: Semantic Image Segmentation. Built a U-Net from scratch to segment the Stanford Background Dataset. Compared different loss functions (Cross-Entropy, Dice Loss, and combined CE+Dice) to handle class imbalance.
